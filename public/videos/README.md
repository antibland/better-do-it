# Weather Video Backgrounds

This directory should contain royalty-free, optimized video files for weather backgrounds:

- `hot-weather.mp4` - Video for hot weather conditions (e.g., sunny, warm scenes)
- `rain-weather.mp4` - Video for rainy weather conditions (e.g., rain, clouds)
- `cold-snow-weather.mp4` - Video for cold/snowy weather conditions (e.g., snow, winter scenes)

## Requirements

- Format: MP4 (or WebM for better compression)
- Optimized for web (compressed, reasonable file size)
- Royalty-free license
- Loopable content
- Suitable as background (not too distracting)

## Suggested Sources

- Pexels Videos: https://www.pexels.com/videos/
- Pixabay Videos: https://pixabay.com/videos/
- Unsplash Videos: https://unsplash.com/videos

## Example Search Terms

- Hot weather: "sunny day", "summer heat", "warm weather"
- Rain: "rainy day", "rainfall", "storm"
- Cold/Snow: "snowfall", "winter", "blizzard", "cold weather"
